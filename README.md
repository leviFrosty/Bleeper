# Bleeper!

The twitter clone made with React and Firebase.

#### Features:

- 🔑 Authentication
  - Email/Password
  - Google
  - Github Oauth
- ☁️ Real-time Cloud Based Data
- 📸 Image attachments!
- 🧭 Routing / Navigation
- 🏷️ Profile page
- ➕👓✏️✖️ CRUD operations

##### Made with:

- React
- Jest
- Firebase
- Sass
